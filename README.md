# Информационная система управления отелем (HIS)

## Описание задания

Данное приложение представляет собой информационную систему управления отелем с графическим интерфейсом на базе Qt6. Система позволяет управлять номерами, гостями, бронированиями и заселениями, а также отслеживать статистику работы отеля.

## Основной функционал:

### Управление номерами:
- Создание и управление номерами различных типов
- Поддержка различных типов номеров: Standard, Deluxe, Suite, Room8Bed, Room6Bed, Room4BedFemale, TwinJunior, DoubleEconomy, StandardDouble, LoftDouble, FamilyRoom
- Управление статусами номеров (доступен, забронирован, занят)
- Настройка характеристик номеров: количество кроватей, WiFi, мини-бар, балкон, джакузи
- Автоматическое назначение номеров при бронировании

### Управление гостями:
- Добавление и редактирование информации о гостях
- Хранение контактной информации (имя, телефон, email)
- Добавление заметок о гостях
- История бронирований гостей

### Управление бронированиями:
- Создание и редактирование бронирований с указанием количества дней проживания
- Поддержка различных типов тарифов (DynamicRate, NonRefundable, FlexibleStart)
- Установка запланированного времени заселения
- Отслеживание времени бронирования, заселения и выселения
- Отмена бронирований

### Операции с номерами:
- Заселение гостей в забронированные номера
- Выселение гостей с расчетом стоимости проживания
- Отмена бронирований
- Поиск номеров по различным критериям (цена, количество кроватей, удобства)

### Расчет стоимости:
- Автоматический расчет стоимости проживания
- Расчет общей выручки отеля
- Расчет будущей выручки по забронированным номерам
- Расчет фактической стоимости проживания при выселении

### Статистика:
- Визуализация данных о номерах отеля
- Количество доступных, забронированных и занятых номеров
- Общая выручка отеля
- Фильтрация номеров по статусу
